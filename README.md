# Personal Diary with Encryption

A secure, encrypted personal diary application written in Python. All entries are encrypted using AES-256 encryption with PBKDF2 key derivation.

## Features

- 🔐 **End-to-end encryption**: All diary entries are encrypted using AES-256
- 📝 **Create, read, update, delete** diary entries
- 🔍 **Search** through all entries
- 🔑 **Password protection** with password change support
- 📅 **Timestamps** for each entry
- 💾 **Local storage** - your data stays on your machine
- 🛡️ **Secure key derivation** using PBKDF2 with salt
